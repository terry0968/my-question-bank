Firebase 雲端同步教學

如何取得 Firebase 金鑰？

  1. 前往 Firebase Console。
  2. 進入專案後，點擊左上角齒輪 選擇「專案設定」。
  3. 在「一般」頁籤捲動到最下方，找到「您的應用程式」。
  4. 如果是新專案，點擊 `` (Web) 圖示註冊你的 APP。
  5. 複製程式碼中 `const firebaseConfig = { ... };` 大括號內的內容。

重要前置設定 (沒設會失敗)：

  • Authentication：點擊啟用，並在登入方式中開啟「匿名 (Anonymous)」。
  • Firestore Database：點擊建立，並在「規則」中設為允許所有人讀寫 (或測試模式)。
